# 📌 Project Title
## Netflix Movies & TV Shows Data Analysis Dashboard (Power BI)

---

# 🧾 Project Description
This project presents an interactive dashboard analyzing Movies and TV Shows data from Netflix. The dashboard provides insights into content distribution, ratings, genres, country-wise availability, and yearly growth trends using Power BI.

It helps in understanding content patterns and supports data-driven decision-making.

---

# 🚀 Key Features

## KPI Cards showing:
- Total Shows (2K)
- Total Movies (988)
- Total TV Shows (1012)
- Total Countries (8)

## 🎯 Filters:
- Country filter (Canada, France, etc.)
- Content Type filter (Movie / TV Show)

## 📊 Visualizations:
- Rating-wise distribution of shows  
- Genre-wise content analysis  
- Year-wise growth trends  
- Country-wise content map  
- Movies vs TV Shows comparison (Donut Chart)

---

# 📈 Insights
- Most content falls under **TV-MA and TV-14 ratings**  
- **Horror & Documentary** genres dominate the platform  
- Nearly equal distribution between Movies and TV Shows (~50-50)  
- Significant growth observed in recent years  
- Content is concentrated in major countries like the US, India, and Europe  

---

# 🛠️ Tools & Technologies Used

- Data Visualization
